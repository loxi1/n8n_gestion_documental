Haz esto en WSL Ubuntu
1. Instala dependencias del sistema
Dentro de WSL:
Bash
sudo apt update
sudo apt install -y python3-venv python3-pip tesseract-ocr tesseract-ocr-spa ghostscript qpdf pngquant unpaper
2. Crea un entorno virtual solo para OCR
Yo lo haría en tu home de WSL, no dentro de /mnt/c/...:
Bash
mkdir -p ~/venvs
python3 -m venv ~/venvs/ocrpdf
source ~/venvs/ocrpdf/bin/activate
3. Instala OCRmyPDF dentro del venv
Bash
pip install --upgrade pip
pip install ocrmypdf
4. Verifica
Bash
ocrmypdf --version
tesseract --version
Prueba rápida con uno de tus PDFs
Desde WSL:
Bash
mkdir -p /mnt/c/D/Proyectos/GestionDocumental/n8n-local/storage/files/ocr_tmp/pruebas
ocrmypdf -l spa --force-ocr \
"/mnt/c/D/Proyectos/GestionDocumental/n8n-local/storage/files/inbox/2026/04/posible_documento_20260416204138431_1.pdf" \
"/mnt/c/D/Proyectos/GestionDocumental/n8n-local/storage/files/ocr_tmp/pruebas/guia_ocr_1.pdf"
Si termina bien, luego puedes probar si tiene texto:
Bash
python3 - <<'PY'
from pypdf import PdfReader
pdf = PdfReader("/mnt/c/D/Proyectos/GestionDocumental/n8n-local/storage/files/ocr_tmp/pruebas/guia_ocr_1.pdf")
text = "\n".join((p.extract_text() or "") for p in pdf.pages)
print(text[:3000])
PY
