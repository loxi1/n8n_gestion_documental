Lo que te recomiendo crear
1. Tabla clientes_destino
Para las empresas o consorcios a donde fue emitido el documento.
SQL
CREATE TABLE clientes_destino (
    id BIGSERIAL PRIMARY KEY,
    nombre_oficial TEXT NOT NULL,
    abreviatura VARCHAR(30) NOT NULL UNIQUE,
    ruc VARCHAR(15),
    ruta_windows TEXT NOT NULL,
    descripcion TEXT,
    estado BOOLEAN NOT NULL DEFAULT TRUE,
    creado_en TIMESTAMP NOT NULL DEFAULT NOW(),
    actualizado_en TIMESTAMP NOT NULL DEFAULT NOW()
);
2. Tabla clientes_destino_alias
Porque en los PDFs el nombre puede venir con variantes.
SQL
CREATE TABLE clientes_destino_alias (
    id BIGSERIAL PRIMARY KEY,
    cliente_destino_id BIGINT NOT NULL REFERENCES clientes_destino(id),
    alias TEXT NOT NULL,
    estado BOOLEAN NOT NULL DEFAULT TRUE,
    creado_en TIMESTAMP NOT NULL DEFAULT NOW()
);
Ejemplos:
BBTI S.A.C.
BBTI SAC
BB TECNOLOGIA INDUSTRIAL S.A.C.
CONSORCIO ILUMINACION TARMA 2025
3. Tabla tipos_documentales
Para no dejar tipos “sueltos”.
SQL
CREATE TABLE tipos_documentales (
    id BIGSERIAL PRIMARY KEY,
    codigo VARCHAR(20) NOT NULL UNIQUE,
    nombre VARCHAR(100) NOT NULL,
    abreviatura VARCHAR(20) NOT NULL,
    descripcion TEXT,
    estado BOOLEAN NOT NULL DEFAULT TRUE
);
Ejemplos:
factura / FACTURA
guia / GUIA
orden_compra / OC
nota_credito / NC
requerimiento_compra / REQ
4. Tabla parametros_documentales
Para reglas como prefijo 04.
SQL
CREATE TABLE parametros_documentales (
    id BIGSERIAL PRIMARY KEY,
    codigo VARCHAR(30) NOT NULL UNIQUE,
    valor TEXT NOT NULL,
    descripcion TEXT,
    estado BOOLEAN NOT NULL DEFAULT TRUE,
    creado_en TIMESTAMP NOT NULL DEFAULT NOW(),
    actualizado_en TIMESTAMP NOT NULL DEFAULT NOW()
);
Ejemplos:
prefijo_factura_oc = 04
ruta_base_windows = \\servidor\compartido
usar_anio_en_ruta = 1
