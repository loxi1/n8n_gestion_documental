Regla
Tienes que usar dos Merge reales y configurarlos para combinar por posición.
Flujo correcto
Plain text
Code original
 ├── Write File
 ├── Guardar_Correo
 │     └── Code_correo_id
 └── Merge_1
Merge_1 debe unir:
entrada 1: Code original
entrada 2: Code_correo_id
Así el resultado de Merge_1 tendrá:
metadata original
correo_id
Luego:
Plain text
Merge_1
 └── Guardar_Documentos
       └── Code_documento_id
Y después:
Merge_2 debe unir:
entrada 1: Merge_1
entrada 2: Code_documento_id
Así el resultado final tendrá:
metadata original
correo_id
documento_id
Y recién eso va a Guardar_Archivos.
