Estructura correcta
Plain text
Code original
  ├─→ Write File
  └─→ Guardar_Correo
         └─→ Code_correo_id
Luego:
Plain text
Code original ───────┐
                     ├─→ Merge_1
Code_correo_id ──────┘
Después:
Plain text
Merge_1 ─→ Guardar_Documentos ─→ Code_documento_id
Y luego:
Plain text
Merge_1 ───────────────┐
                       ├─→ Merge_2
Code_documento_id ─────┘
Finalmente:
Plain text
Merge_2 ─→ Guardar_Archivos
Cómo configurar los Merge
Usa el nodo Merge de n8n en modo Combine by Position.
Como cada PDF genera un item y cada insert devuelve un item correspondiente, “by position” es el modo más simple y adecuado aquí.
