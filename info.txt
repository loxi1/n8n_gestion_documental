🧠 🧩 5. Cómo asignar correlativo mensual (CRÍTICO)
Esto es lo que te faltaba cerrar 🔥
✔ Solo la FACTURA genera correlativo
SQL
SELECT COALESCE(MAX(correlativo),0)+1
FROM documentos
WHERE EXTRACT(MONTH FROM fecha_emision) = 4;
✔ Luego guardas:
SQL
UPDATE documentos
SET correlativo = 1
WHERE id = 2;
✔ Luego todos los relacionados usan ese mismo correlativo
SQL
UPDATE documentos
SET correlativo = 1
WHERE correo_id = 1;
